
# Huffman 编/译码系统（C语言实现）

## 📌 项目介绍
本项目为数据结构课程设计，使用 C 语言实现哈夫曼树的构建、编码、译码功能，支持文件读写与可视化输出。

## 💡 功能简介
- `I` 初始化字符集并构建哈夫曼树
- `E` 对文件内容进行哈夫曼编码
- `D` 对编码文件进行译码还原
- `P` 打印编码文件（50位一行）
- `T` 可视化显示哈夫曼树
- `Q` 退出程序

## 🛠 使用方法

```bash
make
./hfm

### 输入格式
```bash
 A 0.01 //字符前需有一个空格
 b 0.1
 ToBeTran.txt 里写入需要进行编码的文本

```
### 输出的文件
```bash
hfmTree:生成的字符与码字的映射表
CodeFile:ToBeTran编码后的文本
TextFile:CodeFile译码后的文本
CodePrin:ToBeTran编码后的紧凑文本
TreePrint:储存哈夫曼树
```
